StorageConfig

StorageConfig class

Configuration options for IPFS storage.

Controls storage paths, size limits, and garbage collection settings.

Example:

final config = StorageConfig(
  baseDir: '/var/ipfs',
  maxStorageSize: 50 * 1024 * 1024 * 1024, // 50GB
  enableGC: true,
);
