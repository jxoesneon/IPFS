# IPFS CLI Dashboard (Project Neo)

A "hacker-style" terminal interface for controlling `dart_ipfs`.

## Features
- **Matrix UI**: Raw ANSI output for a cyberpunk feel.
- **Zero Dependencies**: Runs on the pure Dart VM (no Flutter/Xcode required).
- **Interactive**: Keyboard controls for node operations.

## Running the App
```bash
dart pub get
dart run bin/main.dart
```

## Controls
- **[S]** Start/Stop Node
- **[A]** Add Demo File
- **[Q]** Quit
